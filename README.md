# Stock Analysis and Prediction App

This is a multi-page Streamlit application for analyzing historical stock data and predicting future prices using an ARIMA model.

## Pages

* **Stock Analysis:** View historical charts (Candle and Line) with technical indicators (RSI, MACD, Moving Average).
* **Stock Prediction:** Get a 30-day price forecast using an ARIMA time series model.

## How to Run

1.  Clone the repository.
2.  Install the dependencies: `pip install -r requirements.txt`
3.  Run the app: `streamlit run Trading_app.py`
